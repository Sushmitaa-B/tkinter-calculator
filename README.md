🧮 Tkinter Calculator

A simple and interactive Graphical User Interface (GUI) Calculator built using Python’s Tkinter library.
This project performs basic arithmetic operations with a clean, button-based interface.

## 🎥 Demo Video

https://github.com/Sushmitaa-B/tkinter-calculator/blob/main/demo/demo.mp4

The video demonstrates basic arithmetic operations like addition, subtraction, multiplication, and division using the GUI.

✨ Features
➕ Addition
➖ Subtraction
✖ Multiplication
➗ Division
🧼 Clear (AC) functionality
🖱️ Interactive button-based GUI
⚡ Real-time input display

🛠️ Tech Stack
Python 🐍
Tkinter (Built-in GUI library)

🚀 How to Run
Clone this repository:
git clone https://github.com/Sushmitaa-B/tkinter-calculator.git
Navigate to project folder:
cd tkinter-calculator
Run the application:
python calculator.py
